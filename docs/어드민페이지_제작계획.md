## 고려 사항

| 항목 | 내용 |
|---|---|
| 최초 ADMIN 계정 | `UPDATE users SET role='ADMIN' WHERE email='ksoung140w@gmail.com'` |
| 자기 자신 보호 | AdminUserService에서 adminId == targetId 체크 구현됨 |
| IP 차단 성능 | Redis Set 캐싱 + @PostConstruct DB→Redis 동기화 구현됨 |
| 문의 이메일 발송 제거 | 기존 sendContactEmail() 제거, sendContactReplyEmail()로 전환됨 |
| 패치노트 마크다운 | 작성: textarea, 보기: react-markdown 또는 white-space: pre-wrap |
| Contact 경로 변경 | `/api/contact` → `/api/contacts` (프론트 contact.ts 수정 필요) |
| IDE 진단 오류 | 새 파일 저장 직후 Spring Security/Redis import 오류는 IDE 인덱싱 타이밍 이슈 (실제 빌드 무관) |
