# 패치노트 아카이브

---

## v0.5.0 | 2026-04-21

### 스도쿠 신규 추가
**[업데이트]**
- 스도쿠 게임이 추가되었습니다
- 난이도: 초급 / 중급 / 고급
- 서버에서 퍼즐을 생성하고 정답을 검증하여 점수 조작 방지
- 메모 기능: 칸에 후보 숫자를 작게 표시 (단축키: N)
- 힌트 기능: 게임당 3회 사용 가능 (랭킹 점수에 영향 없음)
- 점수: 풀이 시간 기반 서버 계산 (초급 최대 1,000점 / 중급 2,000점 / 고급 3,500점)

---

### 게임 점검 상태 관리 기능 추가
**[업데이트]**
- 어드민 페이지에서 게임별 점검 상태를 on/off 할 수 있습니다
- 점검 중인 게임은 목록에 🔧 점검 중으로 표시되며 접근이 제한됩니다
- 어드민 계정은 점검 중에도 모든 게임에 접근 가능합니다

---

## v0.4.0 | 2026-04-20

**[버그수정]**
